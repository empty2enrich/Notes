# Numpy 基础

## 基础操作

### np.where

寻找特定值的小标，同时可以进行替换

```
# 寻找某个值的下标
a = np.random.random((2, 3))
print(a)
print(np.where(a < 1))
print(np.where(a == 1))

<!--输出：-->
<!--[[0.55511209 0.19298066 0.12170198]-->
 <!--[0.2728976  0.94515091 0.80528179]]-->
<!--(array([0, 0, 0, 1, 1, 1], dtype=int64), array([0, 1, 2, 0, 1, 2], dtype=int64))-->
<!--(array([], dtype=int64), array([], dtype=int64))-->

# 寻找替换
a = np.random.random((2, 3))
print(a)
print(np.where(a < 1, 1, a))
print(np.where(a == 1, 1, a))

<!--输出-->
<!--[[0.6834205  0.62282232 0.39461944]-->
 <!--[0.49410505 0.89629302 0.70945752]]-->
<!--[[1. 1. 1.]-->
 <!--[1. 1. 1.]]-->
<!--[[0.6834205  0.62282232 0.39461944]-->
 <!--[0.49410505 0.89629302 0.70945752]]-->
```
